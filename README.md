This project uses sorting algorithms to create a bucketed prioritized task ordering system to make time/task management easier. 
Using the foundation of the Eisenhower matrix, there are 4 buckets called: 
- Important and Urgent
- Important and Not Urgent
- Not Important and Urgent
- Not Important and Not Urgent


Build instructions: 
To run the project: 
1) python app.py
2) Open http://127.0.0.1:5000 in browser
3) Add and delete tasks as preferred
